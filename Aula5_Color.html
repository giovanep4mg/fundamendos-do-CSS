Color = define a cor do texto "html"

Pode definir todo documento.
Exemplo, usando a tag "html" vai deixar todo texto dentro da tag azul "blue":

html{
-------color: blue;
}

Como fica na página:


RGBA e RGBA

A melhor forma é usar o RGB ou RGBA .
Que faz uma mistura com as cores, criando 16 milhões de cores.
Exemplo de código, usando o RGB :

html{
------color: rgb(16, 160, 35);
}

Como fica na página:



RGBA

RGBA = O "A" significa "alfha" , que seria a opacidade da cor.
Exemplo de código, usando o RGBA :

html{
------color: rgba(16, 160, 35, 0.514);
}

Como fica na página:



HEXAdecimal

Hexadecimal = usa numeros para representar as cores, que o total dá 16 milhoes de cores.
Exemplo de código, usando o Hexadecimal :

html{
------color:#2d2149;
}
