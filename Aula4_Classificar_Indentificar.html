Classificar e Indentificar elementos "tag"

Podem ser classificadas e indentificadas, pela "tag", pela "class" e pelo "id"
A tag em si já indentica, mas pode ser indetificado uma diferente da outra, criando uma classe.
Classe é para indentificar elementos com propriedades e caracteristas comuns.
Podem haver várias classe, exemplo:
< h2 class="primeira-classe segunda-classe" >< /h2 >

Temos também o "id" , que é único, pode ser usado várias vezes, porém, somente um em cada página.
Exemplo, com id:
< h2 class="primeira-classe segunda-classe " id="titulo-exemplo" >< /h2 >

Como usar o "Seletor"

Agora, que já tem "seletores" expecifico, vamos ver como modifica-los.
Você seleciona uma "tag", e tudo o que estiver dentro dela será modificado
* Pode ser sobreescrita*
h2 {
----color: red; << vai deixar em vermelho o texto dentro da tag "h2"
}


Para indentificar uma classe, usa o ponto "."
Exemplo de como fica o código:

.titulo-aula{
----color: red; << vai deixar em vermelho o texto dentro da tag "h2"
}


Para indentificar um "id"
Exemplo de como fica o código:

#titulo-primeira-aula{
-------text-decoration: none;
}


Cascada, regras de como vai agir as regras dos elementos

Como serão aplicada as regas que foram colocadas
Vai seguindo uma ordem:
TAG = se não houver nenhuma regra, ela vai coninuar a mesma configuração, pode ser sobreescrita.
CLASS = vai ser aplicada todas as regras ja configuradas nela .
ID = será aplica as regras ja configuradas para ele.

De acordo com o Local onde está o CSS

Há também uma ordem, de acordo com o local onde está o CSS.
EXTERNO = tipo a página criada "style.css" primeiro aplica essas regras, substituindo até as outras já colocadas.
TAG STYLE = aplica as regras que estão aqui.
INLINE = aplicaria as regras colocadas nas linhas.
