Como fica na página:


content-box

Para você acessar outro local, dentro ou fora da sua página.
tem que estar dentro do "section".
width =defini a largura.
height = defini a altura.
O código:
.box{
----box-sizing: content-box;
----width: 10rem;
----height: 5rem;
}

Como fica na página:



padding

O espaço entre a borda e o conteúdo.
Pode ser definido o tamanho para os quatros cantos, começa no sentido horário.
Para as mudanças acontecerem no padding, tem que usar o border-box.
Se estiver "content-box" as mudanças de tamanho, vai ser para ele.
O código:
.box{
----box-sizing: border-box;
----width: 10rem;
----height: 5rem;
----padding: 5px 3px 5px 10px;
}


border

De dentro para fora.
tem que informar; a largura, altura e a cor da borda.
O código:
.box{
----box-sizing: content-box;
----width: 100px;
----height: 100px;
----border: 3px solid black;
}

Como fica na página:


"width"

Define a largura da borda.
Tem várias formas de definir a largura das bordas.
Até tirar a borda "border-bottom: 0px;"
Exemplo de um código:
.azul{
----background-color: blue;
----border-width: 10px;
}

Como fica na página:


"border style "

Define o estilo das bordas.
Exemplo de um código:
.azul{
----background-color: blue;
----border-width: 10px;
----border-bottom-style: double;
}

Como fica na página:


"border radius"

Define o raio da borda, faz uma curva na borda.
Exemplo de um código:
.azul{
----background-color: blue;
----border-width: 10px;
----border-bottom-style: double;
----border-radius: 0 0 50px 10px;
}

Como fica na página:


"margin"

Define o espaço entre a borda e a parte externa.
Exemplo de um código:
.azul{
----background-color: blue;
----margin-bottom: 50px;
}
