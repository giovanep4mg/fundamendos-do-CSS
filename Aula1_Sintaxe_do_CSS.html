SINTAXE DO CSS

Como faz para declarar o CSS :

O CSS é dividido em três termos: Instrução, Seletores e Blocos
Cada "bloco" é uma "instrução", onde vai modificar o que foi selecionado pelo "seletores".
A sintaxe, seria assim:


Seletor {
----------propriedade: valor ;
}

Seletor Seleciona qual é a tag, que irá receber as modificações
propriedade: valor ; É a instrução, como será modificado a "tag"
{ } Abre e fecha o bloco, o bloco seria todo o código CSS
