Text Indent

Defini a indentação do texto, um espaço antes de começa a primeira linha do paragráfo.
O código:
main p{
-------color: rgba(31, 28, 179, 0.514);
-------text-indent: 2rem;
}

Alinhamento do Texto

Pode ser definido em tres tipos:
Center = para centralizar o texto.
Left = o texto fica a direita.
Right = o texto fica a esquerda.
O código :
main p{
-------color: rgba(31, 28, 179, 0.514);
-------text-align: center
}

Text Decoration

Não tem muito suporte em muitos navegadores, tem três principais.
Underline = texto sublinhado.
Line-through = deixa um sublinhado acima do texto.
Overline = faz um risco no meio do texto.
O código :
main p{
-------color: rgba(31, 28, 179, 0.514);
-------text-align: center
}

Letter Spacing

Defini o espaço entre as letras das palavras no texto.
Pode usar pixel, em, rem para adicionar o tamanho do espaço.
O código :
main p{
-------color: rgba(31, 28, 179, 0.514);
-------letter-spacing: 0 rem
}

Text Trasform

defini o tamanho dos caracteres do texto.
capitalize = deixa a primeira letra de toda palavra maiúscula.
lowercase = deixa todo texto com letras minúscula.
uppercase = deixa todo o texto com letras maiúscula.
O código :
header h1{
-----color: blue;
-----text-transform: capitalize;
}
#titulo-primeira-aula{
-----color: black;
-----text-transform: lowercase;
}
.texto-vermelho{
-----color: rgb(6, 99, 44);
-----text-transform: uppercase;
}
